# Real Estate App Template

Base project para una aplicación inmobiliaria construida con Next.js 16.

## Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- ESLint
- React Query
- xlsx
- jspdf
- jspdf-autotable
- react-calendar

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
- `npm run typecheck`
