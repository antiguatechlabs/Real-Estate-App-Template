# Overview

## Que es este proyecto

`real-state-app-template` es una base frontend reusable para aplicaciones inmobiliarias centradas en venta de lotes, operacion comercial y visibilidad ejecutiva.

No representa una empresa puntual ni un producto cerrado. Representa un template serio para arrancar rapido nuevas implementaciones por empresa dentro de un ecosistema mayor.

## Objetivo del template

Este template existe para evitar que cada nueva app inmobiliaria empiece desde cero.

La meta es entregar una base consistente con:

- routing publico y administrativo
- layout enterprise reutilizable
- modulos iniciales ya estructurados
- capa de servicios para integracion con BFF o backend
- tipos de dominio alineados al negocio
- convenciones claras para crecer sin degradar el orden

## Industria objetivo

El template esta pensado para operaciones inmobiliarias orientadas a:

- venta de lotes
- seguimiento comercial de prospectos
- control de proyectos y disponibilidad
- vendedores y equipos comerciales
- pipeline de negocios
- landings por proyecto
- reportes y dashboards administrativos

## Alcance inicial

La version inicial incluye:

- Next.js 16 con App Router
- TypeScript
- Tailwind CSS
- React Query
- placeholders funcionales de modulos clave
- dashboard base con mocks centralizados
- servicios y endpoints listos para integrarse
- carpeta `DOCS` con contexto tecnico y de negocio

No incluye todavia:

- autenticacion real
- middleware de proteccion
- permisos por rol
- multitenencia real
- integracion completa con BFF o backend productivo
- reportes avanzados
